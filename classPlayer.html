Detailed Description

Player class.

This is a child class which is inherited from the Character class. This class contains all of the parent class's parameters and functions. The class contains an additional function and two parameters, which are the LevelUp(), xp and MaxHp.
With these parameters and function we allow for a specific Player, that when it hits a Character it gains xp. When its xp reaches 100, the levelup happens and the Player's stats will improve.
